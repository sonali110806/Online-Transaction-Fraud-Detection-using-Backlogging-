# Online-Transaction-Fraud-Detection-using-Backlogging-
The goal of the system is to detect fraudulent transactions before they are successfully completed, thereby minimizing financial loss and enhancing user security.

This project is a web-based application designed to demonstrate how online transactions can be monitored, analyzed, and blocked based on fraud risk levels. It includes an admin dashboard for tracking suspicious activities and blocked transactions.

---

 Project Overview
The system simulates online transaction monitoring where:
- Transactions are evaluated based on predefined fraud rules
- High-risk transactions are blocked
- Admin can view fraud logs and analytics via dashboard

This project focuses on **backend logic + admin workflow + UI visualization**.

---

 Features
- Admin login system
- Fraud risk score calculation
- Blocked transaction tracking
- Fraud logs with user & order details
- Admin dashboard analytics
- Role-based access control
